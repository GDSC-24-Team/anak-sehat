<!-- GETTING STARTED -->

## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/GDSC-24-Team/anak-sehat.git
   ```
2. Install required dependencies
   ```bash
   npm install
   ```
3. Enter your environment variables in `.env`
   ```bash
   HOST=<HOST>
   PORT=<PORT>
   ```
4. Run
   ```sh
   npm run start-dev
   ```
